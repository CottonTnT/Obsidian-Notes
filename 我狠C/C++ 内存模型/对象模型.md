# 编译器什么时候合成默认构造函数


> 没有定义任何构造函数，编译器某些特定的情况下就会帮我们定义 (合成) 一个默认构造函数





- 以下情况，均默认类中无任何构造函数

| num     | time     |
|:-----|:-----|
| 1     | 父类带缺省 $^{1}$ 的构造函数   |
| 2     |      |
|      |      |
|      |      |


--- 
$^{1}$  缺省构造函数: 构造函数*没有参数*，或者构造函数的*所有参数都有默认值*, *一个类中只能有一个缺省构造函数*.
